PROJECTS

DOWNBEAT

For the final project in our digital signal processing lab, my team designed and prototyped a tool for drummers to transcribe their musical ideas. Downbeat processes a single audio input of a drum kit then generates and displays the corresponding sheet music in real time. The drum classification is implemented in C on an STM32 Nucleo microprocessor and the transcription is implemented in Python on a Raspberry Pi 4.

GESTURE-BASED MIDI SEQUENCER

In our embedded systems class my team developed and built an electronic MIDI sequencer. We traded out the typical knobs and buttons controls for distance sensors and accelerometers to implement a machine with controls that parallel the intuitive gestures used when playing a classic musical instrument.

VOCAL HARMONIZER

I created a vocal harmonizing tool capable of harmonizing with singers during live performances. The algorithm windows an vocal audio track, extracts the relevant frequencies that maintain the voice’s timbre, and pitch shifts the audio by harmonic ratios to generate a robotic chorus. The original MATLAB project has been extended to C code and exported as a VST to be used as a plugin in modern DAWs (digital audio workstations).

CIRCUIT SORT AND SIMULATOR

I created this C++ project in my logic circuit synthesis and optimization course to illustrate the functional simulation of digital circuits. The 2-part algorithm tkaes a circuit written in blif file form as input, then generates a topological ordering of the circuit and simulates the output propogation for a given input vector.

ASSEMBLER

Over the duration of my computer organization course, I wrote programs to simulate the ‘LC-2K’, an 8-register, 32-bit computer with 64kB memory that is capable of solving complex problems. This specific C algorithm turns assembly code into machine code based on the LC-2K instruction set architecture.

RPN CALCULATOR

For my programming and data structures course I designed a reverse polish notation calculator supporting basic operations on integers and floats. The algorithm is implemented with a doubly-linked list functioning as the calculator’s stack.
